Remove Spokespeople Names

This tool removes specified spokesperson names from the Campaign Name column in your CSV file.

Important: Names at the beginning of Campaign Name will NOT be removed.

For example, if your CSV has entries like:

Campaign Name
123 Greeting - Sean Hannity
123 Greeting (Ben Stein)
123 Greeting [Chuck Norris]
123 Greeting Hannity
Sean Hannity - 123 Greeting

It will be processed to:

Campaign Name
123 Greeting
123 Greeting
123 Greeting
123 Greeting
Sean Hannity - 123 Greeting

Note that "Sean Hannity - 123 Greeting" remains unchanged because the name is at the beginning.

Current list of names to remove:

  • Sean Hannity
  • Hannity
  • Ben Stein
  • Dennis Quaid
  • Chuck Norris
  • Mike Lindell

  1. Upload your CSV file using the button below
  2. Click "Process CSV" to remove spokesperson names
  3. Download the processed file
